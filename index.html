About Me

Hello! I'm Hemstone Omollo, a Kenyan Full Stack Developer and founder of Voltageitlabs, a web development agency in Kenya. With over 6 years of experience in software development, I specialize in building scalable, AI-powered web applications using Python/Django and Vue.js/Nuxt.js.

I've successfully built and launched multiple production applications, including chasemyinvoices.com, an AI-powered invoicing SaaS platform currently serving paying customers. My expertise spans the entire software development lifecycle from architecture and database design to deployment and maintenance.

Python has been my primary programming language since 2017, and I've built most of my projects using Django as the backend framework. I'm proficient in Django REST Framework for API development, and I use modern frontend technologies like Vue.js, Nuxt.js, and Tailwind CSS to create responsive user interfaces.

I hold a Bachelor of Science in Mathematics and Computer Science from Kenyatta University. Beyond development, I'm passionate about AI integration, having worked extensively with OpenAI GPT-4 and Claude APIs to build intelligent, agentic systems.
